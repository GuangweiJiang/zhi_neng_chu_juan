
智能出卷系统（2.0版）使用说明

1、 软件简介：
智能出卷系统（2.0版）是一套开放式的试题综合管理和智能化出卷系统，本软件集题库的创建、录入、管理、维护于一体，试题选取灵活快速，试卷编排由电脑直接完成，最大限度地提高学校及其他教育教学单位的题库维护和出卷效率。

2、 适用对象：
适用于各类大、中、小学校及其他教育教学单位， 有利于实现教、考的分离。

3、 适用系统：
Windows 98、Windows Me、Windows 2000、Windows Xp。

4、 系统要求：
须预先安装Word。

5、 安装说明：
点击光盘下的setup即可安装。
